# Sistema de Fazenda em Java

Projeto desenvolvido em Java para estudo de Programação Orientada a Objetos.

O sistema simula o gerenciamento básico de uma fazenda, permitindo cadastrar animais, pastos e currais, movimentar animais entre locais e aplicar remédios,
tudo por meio de um menu interativo no terminal.

Funcionalidades

- Cadastro de animais (Boi, Vaca e Cavalo)
- Cadastro de pastos
- Cadastro de currais com controle de capacidade
- Movimentação de animais entre fazenda, pastos e currais
- Aplicação de remédios em animais
- Visualização da estrutura da fazenda
- Listagem de animais por pasto e por curral
- Sistema interativo via terminal

Conceitos Utilizados

- Programação Orientada a Objetos (POO)
- Herança
- Polimorfismo
- Encapsulamento
- Classes abstratas
- Interfaces
- Listas (`ArrayList`)
